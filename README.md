# zerenadiaz.com

website for [zerenadiaz.com](http://zerenadiaz.com) built using [DocPad](http://docpad.org).

## how to

### install

```
git clone https://github.com/zerenadiaz/zerenadiaz.com
npm install
```

### run

`npm start`

### develop

`npm run develop`

### deploy

`npm run deploy`
